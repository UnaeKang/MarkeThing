# 프로젝트 개요
- 서울시 뉴딜 일자리 클라우드 활용 Java 개발자 양성 및 인턴연계 교육과정
- 주제 : 전통시장 장보기 대행 서비스
- 기간 : 2024.07.01 ~ 2024.07.16
- 인원 : 5명

# 기술 스택

## 프론트엔드
- JSP, jQuery
- Ajax

## 백엔드
- Java 11
- Spring Boot(2.7.x)
- ORM: JPA
- 빌드: Gradle
- AUTH: Spring Security, JWT
- RDBMS: MySQL
- 임시 데이터 저장/캐싱: Redis
- API TEST: Postman
- CI/CD: Docker, Jenkins
